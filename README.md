# IoT Surface Analyzer + Fuel Forecast Dashboard

## 📊 Overview
An interactive Streamlit dashboard that:
- Simulates IoT sensor data (surface analyzer)
- Forecasts fuel prices using ARIMA & LSTM
- Provides scenario testing and policy insights
- Includes gamified prediction and executive summary generator

## 🚀 How to Run
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/surface-analyzer.git
pip install -r requirements.txt
streamlit run app.py
🖥️ Features
Real-time simulation

Gamified prediction game

Scenario testing for policy
